# 💰 SaldoMais App  
_App Flutter para controle de despesas pessoais_

[![Made with Flutter](https://img.shields.io/badge/Flutter-Framework-blue.svg?logo=flutter)](https://flutter.dev)
[![Dart](https://img.shields.io/badge/Dart-Language-0175C2?logo=dart)](https://dart.dev)

[![Udemy Project](https://img.shields.io/badge/Udemy-Curso%20Flutter-orange?logo=udemy)](https://www.udemy.com)

Desenvolvido durante o curso de Dart/Flutter na Udemy como projeto prático de aprendizado.

---

## ✨ Funcionalidades

✅ Adicionar, listar e remover transações financeiras  
📊 Visualizar gráfico das despesas dos últimos 7 dias  
📱 Interface intuitiva, moderna e responsiva  
📭 Feedback visual quando não há transações cadastradas

## ▶️ Como Executar

1. 🔁 Clone o repositório:
   ```bash
   git clone https://github.com/FelipeMourah/SaldoMais.git
   cd expenses

2. Instale as dependências:
   ```sh
   flutter pub get
   ```
3. Execute o aplicativo:
   ```sh
   flutter run
   ```
